❓ שאלות נפוצות

האם זה בטוח?

▼

כן! כל הודעות מוצפנות בFernet, סיסמאות מוגנות בbcrypt, וזה עובד רק ברשת המקומית (אא"כ מוגדר אחרת).

צריך Python?

▼

לא! הקובץ EXE עומד בעצמו. פשוט הורד והפעל.

איך מתחילים?

▼

1. הורד את controlit.zip
2. הפעל על Admin machine וחר ADMIN
3. הפעל על User machine וחר USER
4. בעוד 3-5 שניות אתה רואה את המחשב

צריך Relay server?

▼

לא, לא בצורך בLAN mode. Relay צריך רק אם רוצים בקרה מחוץ לרשת המקומית.

רוצה להוסיף feature?

▼

כל הקוד פתוח! צור fork בGitHub או צור issue עם הצעתך.
